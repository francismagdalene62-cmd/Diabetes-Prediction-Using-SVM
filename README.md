# Diabetes Prediction using Support Vector Machine (SVM)

# Project Overview
This project predicts whether a patient is diabetic using the PIMA Indians Diabetes Dataset and a Support Vector Machine (SVM) classifier.

# The workflow includes:
Data Collection
Data Preprocessing
Feature Standardization
Train-Test Split
Model Training using SVM
Model Evaluation
Building a Predictive System

# Technologies Used
Python
NumPy
Pandas
Scikit-learn
Google Colab

# Dataset
The dataset contains medical information including:
Pregnancies
Glucose
Blood Pressure
Skin Thickness
Insulin
BMI
Diabetes Pedigree Function
Age

# Target Variable:
Outcome (0 = Non-Diabetic, 1 = Diabetic)

# Machine Learning Algorithm
Support Vector Machine (SVM)

# Workflow
Import Libraries
Load Dataset
Data Analysis
Feature Scaling
Train-Test Split
Train SVM Model
Evaluate Accuracy
Predict New Patient

# Sample Output
Prediction: [1]
The person is diabetic
